Projects 2026

The projects designed in 2026

📄️ Theremin

An instrument that plays without you ever touching it.

📄️ DIY Embedded Music Player

A hardware digital audio player built on STM32 using Rust and internal DAC audio processing.

📄️ OxiNMP

Digital Signal Processing on a INMP microphone, having multiple voice effects chosen by the user.

📄️ Breathalyzer

A device that measures the alcohol level in the air used mainly for combating drunk driving.

📄️ Rusty Drone

A compact freestyle drone powered by an STM32 and ESP32 that streams real-time video over WiFi and is controlled via Bluetooth using a PS4/PS5 controller.

📄️ Tricopter Drone

A Real-Time Flight Control System for a Tricopter.

📄️ Thermal Imaging Embedded System (STM32 + MLX90640)

A portable handheld thermal camera that captures, processes, and displays live infrared data in real time.

📄️ RustKey

A hardware TOTP authenticator device with encrypted secret storage, USB-HID output, and an OLED display.

📄️ Secure Autonomous Rover Under Simulated Electronic Warfare

A cyber-resilient autonomous rover system with a TLS-protected control dashboard, live telemetry, and software-only communication fault simulation.

📄️ Smart Air Purifier

An STM32-powered smart air purifier that monitors real-time PM2.5 levels and environmental conditions to automate a 12V HEPA filtration system.

📄️ Scanny, the GuardBot

An autonomous robot that patrols a parking lot and scans license plates.

📄️ Home Environment Station

A touchscreen dashboard that monitors temperature, humidity, light and air quality in real time

📄️ The all-in-one Biometric Hardware Security Key

A fingerprint-authenticated hardware vault that types your passwords and generates live TOTP 2FA codes via USB

📄️ Tamagotchi

A virtual pet that lives, grows, and misses you when you're away

📄️ Musical Glove

A wearable music device

📄️ Orion

An autonomous guide assistant featuring smartphone-linked vector navigation and real-time intelligent disobedience.

📄️ BytArm

A compact embedded robotic arm system powered by an STM32 microcontroller, capable of real-time control of multiple servo motors using analog inputs, designed to explore low-level hardware interaction and precise motion control.

📄️ Rock, Paper, Scissors Robotic Arm

A robotic hand that plays rock, paper, scissors using AI gesture recognition

📄️ RustRover

A Wi-Fi remote controlled Rover that collects, analyzes and stores samples.

📄️ Ferristein 3D - Handheld Game Console

A bare-metal handheld game console, running a custom copy of the renowned classic "Wolfenstein 3D".

📄️ Rusty Coffee

An automated coffee machine in which you can schedule your coffee and customize it to your heart's desire.

📄️ Automated Drink Mixer

An automated drink mixer that precisely measures and dispenses custom juice recipes.

📄️ Smart Dual-Barrier Parking System with NFC and Coin Detection

A smart parking barrier system designed to simulate a realistic automated parking management

📄️ Cash Counting Machine

A device that would count stack of banknotes and display the sum.

📄️ Smart Lockbox

A Rust-based electronic lockbox with keypad PIN access, servo lock control, local feedback, and ESP32 networking for logs/admin reset.

📄️ PicoReflex F1 Edition

A reflex training game built on a Raspberry Pi Pico 2W, housed in an F1-inspired 3D-printed enclosure.

📄️ Project Name

A one line project description

📄️ Autonomous RC Car

An RC car system guided by an external overhead vision module and a stationary STM32 command center.

📄️ Embedded DOOM

A DOOM clone on a microcontroller, controlled via Wi-Fi from your phone.

📄️ Autonomous Context-Aware Thermal Rover

A mobile, dual-axis targeting and navigation system that autonomously patrols an environment, avoids obstacles, and tracks human heat signatures using Rust and the Embassy framework.

📄️ Quadruped Spider Robot

A 4-legged spider robot controlled via Bluetooth, capable of walking in all directions.

📄️ PulseOx

A device that monitors your pulse and oxygen levels, alerting you the moment vitals lose balance

📄️ Energy monitor

A device that analyzes electric power consumption

📄️ SchoolGate

An RFID-based gate system that controls school access, displays student info and logs entry time.
